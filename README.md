# TrianaLogistics

Sistema interno para gerenciar processos de procurement.

**Stack:** Vanilla JS + Supabase

## Funcionalidades

- Upload e validação de BOM
- Cadastro de fornecedores com RFQ
- Matching de cotações com conversão de moeda
- **Base de dados de preços históricos por item/fornecedor**
- **Histórico de performance de fornecedores (tempo resposta, condições pagamento)**
- Geração de Excel formatado
- Controle de acesso por perfil

## Nota

Repo não inclui schema SQL por segurança.
